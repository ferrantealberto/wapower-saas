WAPower N8N Credentials

Configurazione credenziali N8N per autenticazione API WAPower

Credentials Configuration

File di configurazione per le credenziali N8N che permette l'autenticazione con l'API WAPower.

File: credentials/WAPowerApi.credentials.ts

import {
  ICredentialType,
  INodeProperties,
} from 'n8n-workflow';

export class WAPowerApi implements ICredentialType {
  name = 'waPowerApi';
  displayName = 'WAPower API';
  documentationUrl = 'https://docs.wapower.it/api';
  properties: INodeProperties[] = [
    {
      displayName: 'API URL',
      name: 'apiUrl',
      type: 'string',
      default: 'https://api.wapower.it',
      required: true,
      description: 'Base URL dell\'API WAPower',
    },
    {
      displayName: 'API Key',
      name: 'apiKey',
      type: 'string',
      typeOptions: {
        password: true,
      },
      default: '',
      required: true,
      description: 'Chiave API WAPower ottenuta dal dashboard',
    },
    {
      displayName: 'Webhook Secret',
      name: 'webhookSecret',
      type: 'string',
      typeOptions: {
        password: true,
      },
      default: '',
      required: false,
      description: 'Secret per la verifica webhook (opzionale)',
    },
  ];
}

Descrizione Parametri

API URL

URL base dell'API WAPower. Default: https://api.wapower.it

Campo obbligatorio

API Key

Chiave API ottenuta dal dashboard WAPower

Campo obbligatorio, mascherato come password

Webhook Secret

Secret per la verifica HMAC dei webhook

Campo opzionale, mascherato come password

Installazione e Utilizzo

1. Installazione Package N8N

# Installa il package N8N WAPower
npm install n8n-nodes-wapower

# Riavvia N8N
npm run start

2. Configurazione Credenziali

  1. Apri N8N
  2. Vai in "Credentials" â†’ "Create New"
  3. Cerca "WAPower API"
  4. Inserisci API URL, API Key e Secret (opzionale)
  5. Salva le credenziali

3. Utilizzo nei Workflow

Le credenziali saranno disponibili in tutti i nodi WAPower (WAPower Send, WAPower Trigger, etc.)
