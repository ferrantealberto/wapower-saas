Installazione

Installazione tramite NPM:

# Installa il package
npm install n8n-nodes-wapower

# Riavvia N8N
npm run start

Installazione manuale:

# Clona il repository
git clone https://github.com/wapower/n8n-nodes-wapower.git

# Entra nella directory
cd n8n-nodes-wapower

# Installa dipendenze
npm install

# Builda il progetto
npm run build

# Copia nella directory N8N
cp -r dist/* ~/.n8n/nodes/
